# Rejection comment required

> HTTP `400` · `application/problem+json`

**Code:** `rejection-comment-required`

A rejection must include a comment explaining the reason to the author.

## When it occurs

Raised when an Editor or Administrator rejects a content item without providing a
non-empty review comment. The comment is mandatory so that the author receives
actionable feedback and can revise and resubmit.

## How to resolve

Provide a non-empty comment describing why the content was rejected.
