# Resource not found

> HTTP `404` · `application/problem+json`

**Code:** `resource-not-found`

The resource with the provided ID was not found.

## When it occurs

Returned when a requested resource — content, category, tag, user, media, or similar — does not exist for the given identifier or slug, or is not publicly visible.

## How to resolve

Verify the identifier or slug and confirm the resource exists and is accessible to the current user.
