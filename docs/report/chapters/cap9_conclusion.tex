\section{Accomplished Work} \label{sec:conc_accomplished}

This beta report documents the progress achieved in the development of the Diário LX platform. The following components have been implemented or are in an advanced state of development:

\begin{itemize}
  \item System architecture and technology stack definition.
  \item Database schema design and initial migration scripts.
  \item Backend API: authentication, user management, content CRUD, and editorial workflow endpoints.
  \item Frontend: application structure, routing, authentication flow, and initial page implementations.
\end{itemize}

\section{Challenges} \label{sec:conc_challenges}

% TODO: descrever os desafios encontrados
Several challenges were encountered during the beta phase:

\begin{itemize}
  \item \textbf{Rich text editor integration}: Implementing a rich text editor that supports inline images, subtitles, and highlights within the article body proved more complex than initially anticipated.
  \item \textbf{Media storage}: The decision regarding the multimedia storage solution depends on infrastructure availability from IPL, which introduces an external dependency on the project timeline.
  \item \textbf{Editorial workflow}: Correctly modelling and enforcing the full set of state transitions and permission rules required careful design of both the backend and frontend layers.
\end{itemize}
